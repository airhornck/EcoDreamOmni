六、发现的问题
-----------------------------------------------------------
  1. [中] 部分Service层缺少 await db.commit()
     影响: LLM Hub、Proxy等创建后跨请求不可见
     解决: 在 create/register 函数末尾加 await db.commit()

  2. [低] /assets 路由与 nginx 静态目录 /assets 冲突
     影响: 访问 /assets 返回 403
     解决: 将前端路由改为 /asset-pool 或调整 nginx

  3. [低] Proxy Create 返回 404
     影响: 代理配置页面可能无法创建代理
     解决: 检查 proxy API 路由前缀和请求体格式

  4. [低] xhs 库中文编码问题
     影响: 错误消息显示为乱码
     解决: 已在 _extract_xhs_error 中增强处理
