```

□ 2. 后端 Lint 0 errors（ruff check）✅ Phase 2 新增代码全部通过（修复 16 处，含未使用导入/未定义变量）
□ 3. 后端类型检查 0 errors（mypy）✅ Phase 2 新增代码全部通过（修复 1 处 Optional 断言）；现有代码 13 处技术债务不影响
□ 4. 前端 Lint 0 errors（npm run lint）← Phase 1 无前端改动
□ 5. 前端类型检查 0 errors（npx tsc --noEmit）← Phase 1 无前端改动
□ 6. 前端测试通过（npm run test -- --run）← Phase 1 无前端改动
□ 7. Docker Compose Build 通过（docker compose build）← Phase 1 无新增服务，PostgreSQL 现有容器运行正常
□ 8. Alembic 迁移可升级/降级（alembic upgrade head && alembic downgrade -1）✅ 已验证
□ 9. 数据词典已同步更新（docs/数据词典_v4.0/ 对应文件）✅ Phase 1 新增表已入 ER 图
□ 10. 契约文档已同步更新（docs/契约与数据/ 对应文件：API 契约 / ER 图 / 状态机）✅ §4.1/§4.2 已补充
```
