用户使用说明

从知识基建到发布闭环，六步上手素人号矩阵运营

六大基础功能（Function）初始化

在创建任何内容之前，需先完成六大基础功能的数据沉淀。这些库是整个系统的「底层真源」，所有 Agent 生成与审核均通过标准化接口调用，禁止直接操作数据库。基础功能配置为一次性操作，完成后后续每次内容生产自动引用。

1

品牌资料库（BrandKnowledge）

录入品牌故事、Slogan、品类知识、产品 SKU（如驱虫药/猫粮/洗护）、核心卖点、禁用语清单、FAQ。支持向量索引与批量 CSV 导入，ContentForge 生成时自动 RAG 注入。

2

素材库（AssetPool）

三源混合：运营上传真实产品图/场景图（主，≥70%）、授权图库 API 导入（辅）、ImageForge AI 生成封面（补）。每张素材强制记录版权来源（自有/授权/AI生成），打标签关联 BrandKnowledge 产品。

3

时间线库（TimelineLibrary）

维护产品上市时间线、季节营销节点、平台大促日历。将时间线事件与 BrandKnowledge 产品关联（如驱虫季关联驱虫药 SKU），驱动 TrendScout 选题推荐与 CronHub 定时发布。

4

兽药批文库（VetDrugDB）行业真源

录入兽药批文号、有效成分、适应症、用法用量、批准机构、有效期。所有功效宣称必须以 VetDrugDB 为唯一依据，ComplianceGuard 审核时自动校验「宣称 vs 批文」一致性。

5

平台规则库（PlatformRule）风控真源

分层维护：L1 法律红线（广告法/兽药管理条例）、L2 平台静态规则（小红书/抖音/视频号差异）、L3 账号状态规则（新号/老号差异化策略）、L4 动态风控（临时风控/节日策略）。ComplianceGuard 与 Publisher 均引用此库。

6

人设剧本库（PersonaStory）叙事真源

为每个 Persona 定义时间轴驱动的故事剧本（如「新手养猫第1-12周的真实记录」）。配置每期主题、情感曲线、关键事件节点、前情回顾、下期预告。ContentForge 生成时自动注入 persona_story_context，确保内容在宏观故事线中不自洽断裂。

BrandKnowledge 知识条目示例
{
  "entry_type": "product",
  "product_name": "瑞德医生驱虫药（猫用）",
  "category": "驱虫护理",
  "selling_points": ["内外同驱", "幼猫可用", "28天周期"],
  "forbidden_words": ["治疗", "治愈", "替代兽医"],
  "linked_assets": ["asset_cat_001", "asset_cat_002"],
  "linked_timeline": ["deworm_season_2025"]
}

账号池与人设池配置

1

导入账号矩阵

在 AccountPool 中批量导入小红书账号，配置设备指纹、IP 代理与每日发布配额。系统自动计算账号健康分；新号自动进入 warming 策略（低频率阶梯）。

2

创建 Persona（人设）

在 PersonaPool 中定义人设档案：身份背景（如「养猫 3 年的上班族」）、语言风格、专业领域、禁忌话题。支持 AI 辅助生成草案，但须运营确认。

3

创建 PersonaStory 故事剧本 v2.7.2 新增

在 Story Cockpit 中为该 Persona 创建时间轴驱动的故事剧本（如「新手养猫第1-12周的真实记录」）。配置节点序列：每期主题、情绪基调（低落/平稳/高涨/爆发）、关键事件、前情回顾、下期预告。支持拖拽调整节点顺序、预设情感曲线模板（低谷逆袭/渐进成长/悬疑揭秘）、一键复制剧本绑定到其他 Persona。

4

绑定 AIPL/5A 阶段偏好

为每个 Persona 绑定主推方法论阶段（如「AWARENESS 认知期」或「ASK 询问期」），MarketingMethodology 在生成内容时自动匹配对应模板与 KPI 目标。

5

配置 LLM 模型

在 LLM Cockpit 中注册模型（厂家 + 模型名 + APIKey），设定全局默认模型；按需为特定 Agent/Skill 指定覆盖模型（如 ContentForge→GPT-4o，ComplianceGuard→DeepSeek）。节点未指定则继承全局默认。

Persona 人设档案示例
{
  "persona_id": "cat_mom_v1",
  "name": "小橘的铲屎官",
  "bio": "养猫3年，专注新手养猫避坑",
  "voice_style": "口语化、亲切、带emoji",
  "expertise": ["猫咪驱虫", "猫粮选择"],
  "methodology_stage": "AWARENESS",
  "daily_quota": 2,
  "content_series": ["新手养猫100问"]
}

智能选题与任务创建

1

时间线驱动选题推荐

系统根据 TimelineLibrary 当前日期自动推荐选题：如进入「驱虫季」则推荐「猫咪驱虫避坑」系列；进入「618 大促前」则推荐「囤货清单」选题。运营可一键采纳或手动调整。

2

TrendScout 趋势报告

输入关键词（如「幼猫驱虫」），系统自动生成结构化趋势报告：热点条目列表、标题模式分析、话题簇、AIPL/5A 阶段标注。报告可查看、导出、作为选题依据。

3

创建任务并选择工作流

进入 TaskHub，选择预设模板（「标准内容生产」含完整七阶段，或「轻量快速发布」跳过预演）。在 Workflow Cockpit 中可视化查看节点拓扑。任务类型包括：内容生产任务、互动剧本任务。

4

填写 Prompt 变量与图片配置

在 Prompt Registry 白名单变量中填入内容参数：{{topic}}、{{style}}、{{target_audience}}。在 ImageForge 面板中选择封面素材（从 AssetPool 选取或 AI 生成），人工调整封面文字、配色、布局。

5

绑定人设故事剧本（PersonaStory）

为当前任务选择或创建 PersonaStory（如「新手养猫第1-12周的真实记录」）。系统自动注入当前故事节点上下文（前情回顾/本期主题/下期预告/情绪基调），确保内容在宏观故事线中保持连贯。

Agent 工作流自动执行

1

MarketingMethodology 结构分析

自动匹配 AIPL/5A 阶段模板：确定 hook 结构（数字+痛点）、body 结构（干货清单）、cta 结构（互动引导）、disclaimer 结构（合规声明）。同步注入 BrandKnowledge 品牌卖点与禁用语清单。

2

ContentForge 内容生成

按结构模板生成标题+正文，注入 Persona Voice（人设语言风格）、BrandKnowledge RAG 上下文、PersonaStory 故事剧本上下文（当前故事节点/前情回顾/下期预告/情绪基调）。图片配置按 ImageForge 预设执行（素材选取/AI生成/人工模板叠加）。

3

PoolPredictor 互动预演

输出点赞/评论/收藏的预测区间（如 25–60/5–15/8–20）与置信度。冷启动阶段展示「参考区间」文案；达标样本（N≥30）后展示覆盖率考核。

4

ComplianceGuard 四层合规审核

逐层扫描：L1 法律红线（绝对拦截处方药/诊断承诺）→ L2 平台规则（静态关键词）→ L3 账号状态（新号低频率限制）→ L4 动态风控（节日/时段策略）。命中即记录证据链。

人工审核台与单人审核发布

1

进入审核队列

工作流执行至 Publisher 前强制进入 Human-in-the-Loop。审核台左侧展示图文预览（含封面图），右侧展示 Agent 输出摘要（合规结果/预演区间/结构质量分/品牌一致性校验）。

2

三种审核决策

通过：进入发布队列（可选立即/定时）；驳回：任务失败，记录理由进 DLQ；打回修改：指定回退节点（如 ImageForge 换封面、ContentForge 改标题）并修改变量，保留上下文重新执行。

3

单人审核发布（Publisher）

由同一位运营完成内容审核与发布确认。审核记录自动归档，留存 ≥2 年。发布前调用 PlatformRule L3/L4 预检时段合规性与账号日配额。

备注：PRD 建议 Phase 2 演进为「双人复核」（审核员 + 发布确认员双角色），当前 MVP 按原始需求保留单人审核。

数据回流与模型校准

1

数据导入

发布后 T+24h~T+48h，运营通过 CSV/表单导入实际互动数据（点赞/评论/收藏）。连接器为辅，失败自动降级为手动导入，不阻塞后续发帖。

2

昨日战报生成

DataAnalyst 自动聚合：发布篇数、预测区间命中率（覆盖率，仅在 N≥30 子集考核）、平均互动量、环比变化。CES 仅作可选派生指标。

3

异步模型校准

Celery 批任务检查 MAPE 与漂移。超阈值写入「待重训」队列，由人工或定时任务触发训练脚本。禁止秒级在线学习承诺。

4

Prompt 与素材效果追踪

Prompt Registry 统计各版本任务完成率、人工干预率、质量评分；AssetPool 追踪各素材的使用次数与互动表现，辅助运营迭代。

账号管理		 运营 定义人设档案（身份/语言风格/专业领域）；AI 辅助生成草案；绑定 AIPL/5A 阶段偏好；维护 PersonaStory 人设故事剧本（时间轴驱动的故事线） 人设定位文档、对标账号参考（仅公开字段）、故事剧本大纲 Persona 档案 + Voice 模板 + 方法论阶段绑定 + PersonaStory 剧本 人设与账号领域冲突时提示；禁用话题命中时拦截；故事线与时间线冲突时提示

内容生产		 运营+系统 运营输入关键词 + 选择 AIPL/5A 阶段；系统自动生成趋势报告（Mock 数据/手动导入） 关键词 + 阶段过滤 + TimelineLibrary 当前时间线状态 结构化趋势报告：热点条目、标题模式、话题簇、阶段标注、推荐 5A 阶段 无热点数据时提示「扩大关键词范围」；报告生成失败时降级为手动选题
MarketingMethodology 结构分析
内容生产		 系统自动 自动匹配 AIPL/5A 阶段模板；注入 BrandKnowledge 品牌卖点与禁用语；生成合规标签清单 TrendScout 输出 + Persona 人设 + BrandKnowledge RAG 上下文 内容结构模板（hook/body/cta/disclaimer）+ 合规标签清单 + KPI 目标 模板字段缺失时触发 S0 结构不合规告警；品牌卖点不足时提示补充 BrandKnowledge
ContentForge 内容生成
内容生产		 系统自动 LLM 按模板生成标题+正文；注入 Persona Voice；Prompt 变量白名单校验；S0 Schema 结构合规校验 结构模板 + Prompt 变量 + Persona Voice + BrandKnowledge RAG 完整图文内容 + 结构质量分 + 生成 Token 成本 生成超时（默认 60s）触发 TIMEOUT_ALERT；质量分低于阈值时自动 regenerate 一次
