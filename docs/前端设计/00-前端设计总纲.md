### 4.3 六层 Prompt 预览（新增交互）

在 ContentForge 生成过程中，AI Copilot 面板需展示六层 Prompt 的拼接过程：

```
AI Copilot 面板 — 生成进度
├── Layer 1 ✅ PlatformFormat 硬约束（20 字标题限制）
├── Layer 2 ✅ 结构模板（5A 阶段）
├── Layer 3 ✅ 品牌知识（3 条知识片段已注入）
├── Layer 4 ✅ 关键词注入（"新手养猫""驱虫攻略"）
├── Layer 5 ✅ 人设层（省钱狗爸口吻）
├── Layer 6 🔄 风格层（亲切自然 + emoji）
└── 生成中... SSE 流式输出
```

## 七、与 v2.7.x 前端页面的差异

| 维度 | v2.7.x | v4.0 |
|------|--------|------|
| **页面数量** | 9 页 | 10 页（新增 实验室） |
| **全局布局** | 两栏/三栏切换 | Persistent Three-Panel（固定三栏） |
| **AI Copilot** | 弹窗/悬浮球 | 右侧 320px 持久侧边栏 |
| **Inline AI** | 无 | 悬浮在画布元素旁的智能建议 |
| **Action Cards** | 无 | Diff 预览/确认/多选卡片 |
| **Agent 流** | 无 | 底部内嵌 Agent 执行链路 |
| **实验室** | 无 | 全新页面：爆款复刻 → ContentTemplate |
| **六层 Prompt** | 单一大 Prompt | 分层拼接可视化 |
| **主题** | 亮色/暗色 | **浅色模式为默认**（参考 Gemini/Notion，暗色可选） |
