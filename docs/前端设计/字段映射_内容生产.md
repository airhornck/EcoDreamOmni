## 三、Copilot Action Cards → API 映射

| 页面 | Action Card | Action ID | 调用的 API | 工作区同步 |
|------|------------|-----------|-----------|-----------|
| 看板 | ➕ 新建内容 | `quick_create` | `POST /api/task-hub/tasks` | 跳转 `/generate/create` |
| 看板 | 🤖 Agent 推荐 | `agent_recommend` | `GET /api/agents/recommend` | 展示推荐结果 |
| 创建向导 | ✅ 确认创建 | `confirm_create` | `POST /api/task-hub/tasks` | 跳转看板，新卡片出现 |
| 创建向导 | ← 上一步 | `prev_step` | — | Stepper 步骤 -1 |
| 创建向导 | → 下一步 | `next_step` | — | Stepper 步骤 +1 |
| 编辑器 | 💾 保存草稿 | `save_draft` | `POST /api/ai/copilot/save-and-submit` | 保存状态更新 |
| 编辑器 | 🚀 提交审核 | `submit_review` | `POST /api/ai/copilot/save-and-submit` | 状态变为 reviewing |
| 编辑器 | 🔄 重新生成 | `regenerate` | `POST /api/ai/copilot/regenerate-content` | WebSocket 进度 → 内容更新 |
| 编辑器 | ✨ 应用标题优化 | `apply_title` | `PUT /api/task-hub/tasks/{id}/content` | 标题输入框更新 |
