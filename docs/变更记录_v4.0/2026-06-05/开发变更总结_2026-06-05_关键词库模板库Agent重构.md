```
  api/playground.py ──→ KeywordLibraryORM (CRUD + changelog)
  api/content_templates.py ──→ ContentTemplateORM (recommend)

Agent 重构:
  content_forge.py /generate_content ──→ generate_content(keywords=..., template=...)

```
