@router.get("/agents/recommend")
def recommend_agent(platform: str, format: str, persona_id: str = None):
    candidates = filter_agents(platform, format)
    for agent in candidates:
        score = (
            agent.success_rate * 0.40 +
            persona_history_score(persona_id, agent.id) * 0.30 +
            load_balance_score(agent) * 0.20 +
            account_history_score(agent) * 0.10
        )
    return top_agent, alternatives, reason

## 风险与回滚

| 风险 | 缓解措施 |
|------|----------|
| Migration 执行失败 | 生产环境先备份 `tasks` 表 |
| Agent 回填不完整 | 脚本输出未匹配记录，人工补录 |
| 前端后端接口不一致 | API 层同时接受 `agent_id` 和 `workflow_template_id`（二选一），开关控制 |
| 推荐算法性能差 | GIN 索引 + Redis 缓存热点 Agent 列表 |

**回滚方案**:
1. 数据库：`agent_id` 字段 nullable，可随时清空回退
2. API：保留 `workflow_template_id` 兼容分支，配置开关切换
3. 代码：Git tag 标记回滚点 `git tag v4.0-agent-first-{date}`
