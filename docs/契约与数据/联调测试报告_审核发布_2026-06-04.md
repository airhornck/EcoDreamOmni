## 四、环境搭建命令

### 4.1 Docker 快速启动（推荐）

```bash
# 在 EcoDreamOmni/apps/backend 目录下执行
docker run -d \
  --name ecodream-postgres \
  -e POSTGRES_USER=ecodream \
  -e POSTGRES_PASSWORD=ecodream \
  -e POSTGRES_DB=ecodream \
  -p 5432:5432 \
  -v pgdata:/var/lib/postgresql/data \
  postgres:15-alpine

docker run -d \
  --name ecodream-redis \
  -p 6379:6379 \
  redis:7-alpine
```

### 4.2 启动后端服务
