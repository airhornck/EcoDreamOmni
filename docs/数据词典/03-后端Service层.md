|----------|-----|------|
| `CONTENT_CREATION_NOTE_IMAGE` | `content_creation_note_image` | 图文内容生产（8节点） |
| `CONTENT_CREATION_VIDEO_CLONE` | `content_creation_video_clone` | 视频复刻（8节点，含脚本拆解/复刻） |
| `CONTENT_CREATION_VIDEO_ORIGINAL` | `content_creation_video_original` | 视频原创（8节点，含原创脚本/分镜） |
| `CONTENT_CREATION_TEXT_ARTICLE` | `content_creation_text_article` | 长文章生产（8节点，含文章大纲/长文） |

| 新增函数 | 签名 | 说明 |
|----------|------|------|
| `recommend_template` | `(platform_id: str, content_format: str) -> Optional[WorkflowTemplate]` | 根据平台+格式推荐模板 |

### task_hub.py — simulate_node_output 内容格式感知

| 变更点 | 说明 |
|--------|------|
| `simulate_node_output` (content-forge/cf-body) | 读取 `task.content_format`，动态设置 `content_type`（note/video/article/text_only） |
| 视频格式 | 输出新增 `video_script` / `video_duration` / `video_shots` |
| 长文章格式 | 输出新增 `article_outline` / `word_count` |
| 仅文字格式 | 输出新增 `word_count` |

### task_function.py — Task 模型扩展

| 字段 | 类型 | 说明 |
|------|------|------|
| `content_format` | `Optional[str]` | 内容格式：图文/视频/仅文字/长文章/视频复刻/视频原创 |
