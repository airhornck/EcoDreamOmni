## 五、Copilot-Driven 新增表（v4.0 Step 2 冻结）

### 5.1 `ai_cover_generation_jobs`

AI 封面生成异步任务表。

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | VARCHAR(32) PK | 任务 ID，如 `cover_gen_xyz789` |
| `task_id` | VARCHAR(32) FK | 关联 tasks.id |
| `user_id` | VARCHAR(32) | 用户 ID |
| `prompt` | TEXT | 用户输入的提示词 |
| `auto_prompt` | BOOLEAN | 是否自动根据内容生成提示词 |
| `style_preset` | VARCHAR(32) | 风格预设：`cute`/`professional`/`minimal` |
| `count` | INTEGER | 生成数量，默认 2 |
| `ratio` | VARCHAR(10) | 裁剪比例，默认 `3:4` |
| `status` | VARCHAR(20) | `queued`/`generating`/`completed`/`failed` |
| `results` | JSONB | 生成结果 `[{url, thumbnail_url, ratio, seed}]` |
| `error_message` | TEXT | 失败原因 |
| `created_at` | TIMESTAMPTZ | 创建时间 |
| `completed_at` | TIMESTAMPTZ | 完成时间 |

**索引**:
```sql
CREATE INDEX idx_cover_jobs_user_created ON ai_cover_generation_jobs(user_id, created_at);
CREATE INDEX idx_cover_jobs_task ON ai_cover_generation_jobs(task_id);
```

### 5.2 `copilot_context_sessions`

Copilot 上下文会话表，TTL 30 分钟。

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | VARCHAR(32) PK | 会话记录 ID |
| `user_id` | VARCHAR(32) | 用户 ID |
| `session_id` | VARCHAR(64) UNIQUE | 客户端会话 ID |
| `page` | VARCHAR(64) | 当前页面路由 |
| `selected_items` | JSONB | 选中项 ID 列表 |
| `selected_content` | JSONB | 选中内容摘要 |
| `workspace_state` | JSONB | 工作区状态 |
| `suggested_cards` | JSONB | 后端预计算的卡片建议 |
| `created_at` | TIMESTAMPTZ | 创建时间 |
| `updated_at` | TIMESTAMPTZ | 更新时间 |
| `expires_at` | TIMESTAMPTZ | 过期时间，默认 NOW() + 30min |

**索引**:
```sql
CREATE INDEX idx_copilot_sessions_user ON copilot_context_sessions(user_id, updated_at);
CREATE INDEX idx_copilot_sessions_expires ON copilot_context_sessions(expires_at);
```

| `request_payload` | JSONB | 请求体 |
| `response_payload` | JSONB | 响应体 |

### 5.5 `content_regeneration_jobs` 新表（内容生产 Step 2 冻结）

AI 内容重新生成异步任务表。

```sql
CREATE TABLE content_regeneration_jobs (
  id VARCHAR(32) PRIMARY KEY,
  task_id VARCHAR(32) NOT NULL REFERENCES tasks(id),
  agent_id VARCHAR(64) NOT NULL,
  style_option VARCHAR(32),
  length_option VARCHAR(32),
  tone_option VARCHAR(32),
  prompt_variables JSONB DEFAULT '{}',
  status VARCHAR(20) NOT NULL DEFAULT 'queued',
  result_content JSONB,
  error_message TEXT,
  started_at TIMESTAMP WITH TIME ZONE,
  completed_at TIMESTAMP WITH TIME ZONE,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW(),
  updated_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

CREATE INDEX idx_regen_jobs_task ON content_regeneration_jobs(task_id, created_at);
CREATE INDEX idx_regen_jobs_status ON content_regeneration_jobs(status);
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | VARCHAR(32) PK | 任务 ID，如 `job_reg_001` |
| `task_id` | VARCHAR(32) FK | 关联 tasks.id |
| `agent_id` | VARCHAR(64) | 执行生成的 Agent ID |
| `style_option` | VARCHAR(32) | 风格选项：`casual`/`professional`/`humorous` |
| `length_option` | VARCHAR(32) | 长度选项：`short`/`medium`/`long` |
| `tone_option` | VARCHAR(32) | 语气选项：`friendly`/`serious`/`playful` |
| `prompt_variables` | JSONB | 用户传入的 Prompt 变量 |
| `status` | VARCHAR(20) | `queued`/`running`/`completed`/`failed` |
| `result_content` | JSONB | 生成结果（标题/正文/标签等） |
| `error_message` | TEXT | 失败原因 |
