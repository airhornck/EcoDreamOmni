## 四、前后端数据流

```
Frontend (React)
  ├── Zustand Store
  │     ├── aiCopilotStore ──→ POST /ai/conversations
  │     ├── labStore ──→ — (纯前端状态)
  │     ├── playgroundStore ──→ POST /api/playground/analyze
  │     │                        POST /api/playground/template
  │     │                        GET  /api/playground/keywords
  │     │                        GET  /api/playground/categories
  │     ├── agentFlowStore ──→ WS   /ws/pipeline/{id}
  │     └── taskHubStore ──→ REST /task-hub/*
  └── Component
        ├── AICopilotPanel ──→ useSSEStream
        ├── LabPage ──→ labStore + CapabilityNav
        ├── CapabilityNav ──→ labStore
        ├── ViralAnalyzerCapability ──→ playgroundStore
        ├── LockedCapabilityPlaceholder ──→ labStore
        ├── LabPage ──→ ViralAnalyzerCapability / LockedCapabilityPlaceholder
        ├── AgentFlowBar ──→ useAgentFlowStore + WebSocket
        └── InlineSuggestionCard ──→ inlineAIStore

Backend (FastAPI)
  ├── API Layer (src/api/)
  │     └── Router → Pydantic Schema → Service
  ├── Service Layer (src/services/)
  │     ├── workflow_engine.py ──→ core/checkpoint.py
  │     ├── llm_hub.py ──→ core/event_bus.py (ROUTED 事件)
  │     └── task_hub.py ──→ core/checkpoint.py
  ├── Core Layer (src/core/)
  │     ├── checkpoint.py ──→ PostgreSQL + Redis + 本地文件
  │     ├── event_bus.py ──→ Redis Streams / 内存 fallback
  │     └── database.py ──→ AsyncSession (SQLAlchemy 2.0)
  └── Model Layer (src/models/)
        └── ORM → Alembic 迁移
```
