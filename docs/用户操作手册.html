1. 系统概述

🎯 产品定位

EcoDreamOmni 是面向宠物健康领域的素人号矩阵 AI 内容管理与分发平台。通过 AI 辅助内容生成、自动化发布、流量预测与合规风控，帮助运营团队在小红书、抖音、视频号三大平台高效管理多个素人账号矩阵。

2.3 Token 机制

系统采用 JWT Bearer Token 认证：

  • Refresh Token：有效期约 7 天，用于刷新 Access Token

5. 素材库

素材库集中管理图片、视频、文案等创作素材，支持 AI 生成素材与外部采购素材的统一管理。

5.1 上传素材

  1. 左侧导航栏点击「素材库」→「上传素材」
  2. 选择文件或拖拽到上传区域，支持批量上传
  3. 填写素材信息：
    • 素材名称、分类、标签
    • 来源类型（原创 / AI生成 / 采购 / 用户授权）
    • 版权信息（版权方、授权期限）
    • AI 生成需勾选「AI 披露」并填写模型信息
  4. 点击「确认上传」，系统自动保存到存储并入库
⚠️ 版权合规
商业用途素材必须确认版权授权状态。上传采购素材时请填写授权编号和授权期限，避免版权纠纷。

5.3 AI 生成素材

系统支持通过 AI 模型生成图片素材：

  1. 点击「AI 生成」按钮
  2. 输入生成提示词（prompt），描述所需画面
  3. 选择生成参数：尺寸、风格、模型
  4. 点击「生成」，等待 AI 返回结果
  5. 预览生成结果，满意后点击「保存到素材库」
💡 最佳实践
AI 生成素材会自动标记 ai_disclosure=true 和 ai_model 信息。平台发布时会根据目标平台规则自动添加 AI 创作声明。

6. 内容生成

内容生成是平台的核心工作流，支持从选题策划到文案产出的全流程 AI 辅助。

6.1 创建工作流任务

  1. 左侧导航栏点击「内容生成」→「新建任务」
  2. 选择工作流模板（如：小红书图文、抖音短视频、视频号长文）
  3. 配置任务基础信息：
    • 任务名称
    • 目标账号（从账号池选择）
    • 人设故事线（persona story）
    • 内容主题 / 节点
  4. 设置发布计划：立即执行 / 定时发布
  5. 点击「创建任务」
📷 [新建任务表单截图占位]

6.2 AI 文案生成

任务创建后，进入内容编辑页面：

  1. 点击「AI 生成文案」按钮
  2. 系统根据以下参数调用 LLM 生成内容：
    • 账号人设与故事线
    • 品牌知识库（产品信息、合规约束）
    • 平台规则（小红书/抖音/视频号风格差异）
    • 当前节点主题
  3. AI 返回多版文案草稿，运营人员选择最合适的版本
  4. 在编辑器中手动微调文案，调整语气、添加 emoji、优化排版

6.3 关联素材

  1. 在内容编辑页面点击「添加素材」
  2. 从素材库中选择图片/视频，支持搜索和预览
  3. 拖拽调整素材顺序
  4. 为每张图片添加 alt 文本（无障碍与 SEO）

6.4 提交合规审核

内容编辑完成后，点击「提交审核」，系统自动进行多层级合规检测（详见第 7 章）。

7. 合规审核

合规风控是平台的生命线。系统通过三层防御体系自动拦截违规内容，确保 100% 符合平台规则与法律法规。

7.1 三层合规检测

🚫 第一层：违禁词过滤

实时扫描文案中的敏感词、违禁词，按严重程度分级：

  • 严重：处方药、诊断承诺、绝对化用语 → 自动拦截
  • 警告：夸大功效、未标注合作 → 需人工确认
  • 提示：建议性表述 → 标黄提示
📋 第二层：平台规则匹配

按目标平台匹配专项规则：

  • 小红书：禁止站外引流、医疗资质要求
  • 抖音：商品分享规范、虚假宣传判定
  • 视频号：内容分级、版权合规
🤖 第三层：AI 语义审核

调用大模型进行语义理解，识别隐晦违规表达（如暗示疗效、变相广告等），结合向量检索匹配历史违规案例。

7.2 审核结果处理

结果图标处理动作
通过🟢进入发布队列，等待发布
警告🟡提示修改建议，运营人员确认后可强制通过
驳回🔴必须修改后才能重新提交

7.3 审核记录查询

  1. 左侧导航栏「合规中心」→「审核记录」
  2. 查看所有内容的审核历史，包含：
    • 审核时间、审核层级
    • 命中规则详情（哪条规则、匹配文本）
    • 审核决策与原因
    • 人工复核记录
🚨 绝对红线
  • 严禁发布处方药相关推广内容
  • 严禁做出诊断/治疗承诺（如"吃XX治好猫传腹"）
  • 商业合作内容必须标注"合作/体验/赞助"
  • AI 生成内容未披露将被平台处罚

8. 发布管理

发布管理负责将审核通过的内容推送到目标平台，支持即时发布与定时发布。

8.1 发布队列

  1. 左侧导航栏「发布管理」→「发布队列」
  2. 查看待发布任务列表，包含：账号、平台、内容标题、计划发布时间
  3. 可手动调整发布顺序或取消发布

8.2 发布执行

发布模式说明适用场景
即时发布审核通过后立即推送到平台热点追蹤、紧急内容
定时发布按设定时间自动发布日常运营、批量排期
智能排期系统根据流量预测自动选择最佳发布时间追求最大曝光

8.3 发布状态追踪

发布过程中可实时查看状态：

  • 等待中：在发布队列中等待
  • 发布中：正在调用平台 API / 模拟浏览器发布
  • 已发布：内容已成功上线，返回平台帖子 ID
  • 失败：发布异常，可查看错误日志并重试

8.4 发布后数据回流

发布后系统自动拉取平台数据：

  • 点赞数、评论数、收藏数、转发数
  • 阅读量/播放量
  • 互动率、完播率（视频）
  • 数据自动汇总到驾驶舱
💡 发布建议
  • 同一账号建议间隔 ≥ 2 小时 再发下一条，避免平台限流
  • 每日发布量不超过账号每日配额
  • 最佳发布时间：工作日 12:00-13:00、18:00-20:00、22:00-23:00

9. 驾驶舱

驾驶舱提供全局数据看板，帮助管理者实时掌握矩阵运营状况。

9.1 核心指标概览

📈 内容指标
  • 今日/本周/本月发布量
  • 内容审核通过率
  • 平均生成耗时
  • AI 调用 Token 消耗
🌐 流量指标
  • 总曝光量 / 总阅读量
  • 总互动量（赞/评/藏/转）
  • 平均互动率
  • 粉丝净增长

9.2 账号健康度

驾驶舱展示每个账号的健康雷达图：

  • 内容质量分：内容原创度、合规率
  • 活跃度分：发帖频率、互动响应
  • 风控分：违规次数、限流状态
  • 成长分：粉丝增长、爆款率

9.3 趋势图表

  • 发布时间轴：查看未来 7 天的发布排期
  • 流量趋势：对比不同平台、不同账号的流量曲线
  • 内容热力图：识别高表现内容类型

10. 代理配置

代理配置用于管理账号的 IP 环境，每个账号建议绑定独立代理以避免平台风控。

10.1 添加代理

  1. 左侧导航栏「系统管理」→「代理配置」
  2. 点击「新增代理」按钮
  3. 填写代理信息：
    字段说明示例
    代理名称便于识别的名称上海住宅IP-01
    协议HTTP / SOCKS5SOCKS5
    主机代理服务器 IP51.77.190.247
    端口代理服务端口9595
    用户名代理认证用户名（如需要）user01
    密码代理认证密码（如需要）******
    区域模拟地理位置上海
  4. 点击「保存」，系统自动进行连通性检测
💡 代理持久化
代理配置自动保存到 PostgreSQL 数据库，服务重启后自动恢复。无需每次重新配置。

10.2 代理健康检查

系统定时检测代理可用性：

  • 🟢 正常：代理可用，响应正常
  • 🟡 警告：响应慢或偶发失败
  • 🔴 失效：无法连接，需更换

失效代理会在账号详情页中标记提醒，运营人员需及时更换。

14. ⚠️ 合规红线（必读）

🚨 绝对禁止（系统将 100% 拦截）
  1. 处方药推广：任何处方药的宣传、推荐、效果展示
  2. 诊断/治疗承诺：如"吃XX治好猫传腹""3天见效"
  3. 绝对化用语："最好""第一""100%有效"等无法验证的断言
  4. 冒用资质：未取得兽药/医疗资质却暗示专业背书
  5. 站外引流：小红书平台直接引导加微信/QQ
⚠️ 必须标注（不标注将被平台处罚）
  1. 商业合作：收到品牌方产品/费用的内容必须标注"合作/体验/赞助"
  2. AI 生成：AI 生成的图片/文案需明确标注"由 AI 生成"
  3. 风险提示：涉及用药建议需添加"请咨询执业兽医师"
✅ 安全表达范例
❌ 禁止✅ 允许
"这款药治好了我家猫的肾衰""我家猫确诊肾衰后，在医生指导下使用了XX，目前指标稳定"
"最强驱虫药，100%有效""这款驱虫药覆盖XX种寄生虫，我家用下来效果不错"

15. 常见问题

Q1: 登录时提示"邮箱或密码错误"怎么办？

A: 请确认：

  • 使用邮箱而非用户名登录
  • 检查大小写锁定（Caps Lock）
  • 联系管理员确认账户是否被禁用
  • 若忘记密码，联系 admin 重置

Q2: 发布失败，提示"代理不可用"怎么办？

A:

  1. 进入「代理配置」检查对应代理状态
  2. 点击「重新检测」验证连通性
  3. 若代理失效，更换新的代理 IP 并更新到账号配置

Q3: AI 生成文案质量不佳如何优化？

A:

  • 完善人设故事线，让 AI 更了解账号风格
  • 丰富品牌知识库，提供充分的产品信息
  • 调整生成参数：temperature（创造性）和 top_p
  • 尝试切换不同的 LLM 模型

Q4: 内容审核被驳回，如何快速修改？

A:

  1. 查看驳回详情，了解命中了哪条规则
  2. 根据提示修改违规表述
  3. 使用系统推荐的「安全替代表达」
  4. 重新提交审核

Q5: 如何查看账号是否被平台限流？

A:

  • 驾驶舱中查看账号健康分，低于 60 分可能已被限流
  • 对比历史发布数据，若阅读量/互动量断崖式下降，可能被限流
  • 暂停发帖 3-7 天，模拟正常用户行为恢复权重

Q6: Token 过期了需要重新登录吗？

A: 系统会自动使用 Refresh Token 续期 Access Token，正常情况下无需手动重新登录。若 Refresh Token 也过期（超过 7 天未操作），则需要重新登录。
